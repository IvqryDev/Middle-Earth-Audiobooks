# A repository containing audibooks for the original Lord of The Rings trilogy and The Hobbit read by Phil Dragash.

Each chapter of each book is it's own .MP3 file, averaging in at about an hour long. There are compressed .ZIP files available for each book available for download as well.

The Hobbit/There And Back Again is the 2022 remaster version, while the original trilogy is from 2013/2014.

I got these from the Internet Archive, and put them on here for easy of access for myself, and anyone else who would like to listen to these audiobooks.
